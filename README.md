#discription
This dataset analyzes students’ social media usage habits to assess addiction levels and overall impact.
It includes demographic details, usage behavior (such as daily hours and most-used platform), and indicators like sleep, academic performance, and mental health scores.
The goal is to identify patterns of unhealthy usage and support research, analysis, and predictive modeling on digital well-being.
